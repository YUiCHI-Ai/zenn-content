## この本の対象読者

- ターミナルでの開発に慣れているエンジニア
- AIコーディングアシスタントに興味があるが、まだ活用できていない方
- GitHub Copilotは使ったことがあるが、より深い支援を求めている方

## Claude Codeとは

Claude Codeは、Anthropicが提供するターミナルベースのAIコーディングアシスタントです。エディタのプラグインではなく、**ターミナルに常駐するAIエンジニア**として動作します。

主な特徴は以下の3つです。

1. **ファイルシステムへの直接アクセス**: プロジェクト全体を理解した上でコードを読み書きする
2. **bashコマンドの実行**: テスト実行、ビルド、git操作をAIが直接行う
3. **MCP（Model Context Protocol）**: 外部サービス（Notion、Slack、DBなど）と接続して操作する

## なぜターミナルベースなのか

エディタ統合型のAIアシスタント（Copilot、Cursor等）は、開いているファイルの範囲で補完を行います。一方Claude Codeは、プロジェクトのディレクトリ構造全体を把握し、複数ファイルにまたがる変更を一度に行えます。

```
# Copilot: 開いているファイルのコード補完
# Claude Code: プロジェクト全体を理解した上での操作

$ claude "認証ミドルウェアを追加して、既存の全ルートに適用して"
# → middleware/auth.ts を新規作成
# → routes/*.ts の全ファイルを修正
# → テストファイルも対応修正
```
